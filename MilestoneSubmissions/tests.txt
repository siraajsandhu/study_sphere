Register: 
   Positive: 
      There is a unique Username provided.
      There is a Password input aka not null. 
   Negative: 
      If the password returned is null, the test will fail.
      The Username is already being used. 
      Password doesnt meet requirements.

Login:
  Positive:
      Username is successfully found in the data base.
      The hashed password matches the password saved in the database.
  Negative: 
      The user is not found in the database.
      The User is found but the password does not match.
Profile password change:
  Positive:
      The password is successfully altered in the database, erasing the old one.
  Negative:
      The password fails to meet the requirements for a password.
